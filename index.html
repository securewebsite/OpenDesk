OpenDesk is based on international development standards and internet technologies. This means that OpenDesk can be extremely versatile in use and customized to suite specific needs. Modules provide functionality within OpenDesk and can be added or removed depending on the implementation requirements. Additional modules can easily be developed where unique requirements are identified.

OpenDesk is well documented and support is available through our hosting packages. On-Site training workshops and Webinars offer users access to platforms where questions and concerns can be raised and discussed in detail.

OpenDesk is a cloud based client management system and does not require installation on a end-user computer. All functionality and data is stored on web servers which can be accessed through any internet enabled device with the required browser software. Future editions of OpenDesk will be made available via Tablet and Smartphone devices.

OpenDesk has a refreshingly different layout and a clean user experience that is both easy to use and extendable onto mobile platforms. The interface was designed to minimize screen clutter and is visually appealing which invites the user to spend more time within the environment.

OpenDesk allows short-term insurance brokers the ability to document every insurance policy and match them to their respective owners, while attaching all relevant communications and documentation to a single record.
Short-Term Insurance Brokers
OpenDesk enables long-term insurers the ability to record all policy details along with any renewal dates and possible replacement information to each client while at the same time creating reminders and calendar events.
Long-Term Insurance Brokers
OpenDesk has a built in document storage tool which provides cloud storage space for each client listed. This allows for the proper recording of advice records and risk profiles to back up any investment decisions made.
