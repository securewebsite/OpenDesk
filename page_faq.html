Where can I download OpenDesk?

OpenDesk is currently 203MB in size and may take up to 2 hours to download on slower broadband connections. The link below will provide you with a ZIP file which can easily be extracted using any decompression software.

Download OpenDesk

What are the system requirements?

OpenDesk runs on a combination of MySQL, PHP and Apache on either a Linux (LAMP) or Windows (WAMP) configured server. The installation process will guide you through the necessary versions and extensions required. We recommend using a Virtual Private Server (VPS) through WebAfrica or Afrihost in order to get the best mobile experience.
