Features

  • Home
  • Features

Lead Management

Create, track, follow-up and report on new leads entered with a variety of built in lead management functions.

Client Management

Convert existing leads or create new clients immediately and enable contact management from a central point.

User Management

Access levels and profiles for each individual user can be defined and controlled with complete security.

Financial Products

Product selections are completely customizable and allows for all financial products as defined in the FAIS Act of 2002.

Tasks and Reminders

All tasks and tickets can be shared amongst users and reminders are sent out automatically.

SMS Functionality

Clickatell API provides integrated SMS functionality including bulk messages and marketing campaigns.

Email Integration

Full webmail integration is currently under development, but email sendouts are functional and included.

Fully Customizable

All fields within the system are fully customizable. This does however require a level of understanding of web development.

Modular and Extensible

Additional functionality can be added upon request and consultation to fit the specific needs identified.
